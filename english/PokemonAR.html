Pokemon Card AR

Project allowing easy creation of Pokémon cards in augmented reality.

This project was developed in Processing, a language based on Java, which was introduced during the virtual reality course. Following the course, I worked on several projects with this language.

The goal of this project was to create Pokémon cards in augmented reality while allowing easy addition of new cards. To achieve this, I wanted to avoid modifying the existing code every time a new card needed to be added. Now, all that's needed is to create a specific folder containing all the necessary information to integrate a new card.
