BirdCards

BirdCards is an educational application that helps you learn to recognize birds through their names, images, and songs. Built with Python and Flask for the backend and HTML/CSS/JavaScript for the frontend, the app features an intuitive, responsive interface that works on both desktop and mobile devices.

Key features include:

  • Displaying bird names in French or English
  • Viewing images and listening to sounds (Xeno-canto)
  • No-repetition mode until the list is exhausted
  • Filters by difficulty level and media type (image, sound)
  • Light/dark themes and dynamic language switching (FR/EN)

Technologies Used

  • Python (Flask)
  • HTML, CSS, JavaScript
  • AJAX for dynamic updates
